Generated Organization Data Files

This directory contains generated JSON data files derived from a variety of official ASF organizational metadata, and is recreated at least daily by cronjobs run with Python scripts. Technical details are in the scripts/README.txt file, or see the About page for an overview of the whole site.

  • accounts-evolution.json generated by cronjobs/countaccounts.py from LDAP
  • groups.json, people.json and people_name.json generated by cronjobs/parsecommitters.py from https://whimsy.apache.org/public/
  • podlings.json and podlings-history.json generated by cronjobs/podlings.py from http://incubator.apache.org/podlings.xml
  • projects.json imported with cronjobs/parseprojects.py from projects' DOAP files
  • releases.json and releases-files.json generated by cronjobs/parsereleases.py from http://www.apache.org/dist/
  • committees.json and committees-retired.json imported with import/parsecommittees.py from https://svn.apache.org/repos/private/committers/board/committee-info.txt
  • repositories.json written from http://git.apache.org
